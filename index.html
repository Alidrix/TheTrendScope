The Trend Scope

Détecteur personnel des vidéos YouTube les plus virales (FR/US/ES).

YouTube Data API Supabase Auth sécurisée
  • Détection virale avec vélocité (vues/h) et badge Shorts.
  • Filtrage par pays, catégorie et type de contenu.
  • Historique des vues/likes + notes personnelles synchronisées.
  • Notifications internes et rafraîchissement manuel maîtrisé.
  • Accès par identifiants Supabase (stockés côté serveur).

Configuration requise

Ajoute tes clés SUPABASE_URL, SUPABASE_ANON_KEY, SUPABASE_SERVICE_ROLE_KEY et YOUTUBE_API_KEY dans un fichier .env avant de lancer npm run init:db puis npm start.

Connexion sécurisée

Les identifiants sont vérifiés côté Supabase avant d'accéder au tableau de bord.

Mot de passe oublié ?

Ce formulaire prépare la requête Supabase. Mets à jour les identifiants par défaut dans la table public.admins si nécessaire.
